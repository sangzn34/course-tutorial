10 SSH Key · 3 ที่ ทำอะไร

⭐ private key ไม่เคยอยู่ 2 ที่พร้อมกัน — เกิดที่เครื่องเรา ย้ายเข้า Secret จบ; VPS มีแค่ public

Checklist · "self-host เป็นแล้ว"

เทสจริงทุกข้อ ไม่ใช่แค่เขียน

✓ssh ได้ด้วย key เท่านั้น, ufw แค่ 22/80/443
✓image runner ไม่มี .env / source
✓https เขียว + cert ต่ออายุเอง
✓push main → เว็บอัพเดตเอง
✓reboot → เว็บกลับมาเองใน ~1 นาที
✓monitor เตือนเมื่อดับ · rollback sha เก่าได้
⭐ state อยู่ Supabase หมด → server ไม่มี state = พังก็สร้างใหม่ตาม runbook (cattle ไม่ใช่ pet)

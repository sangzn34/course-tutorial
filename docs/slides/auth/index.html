09 Core Concept

Authentication ≠ Authorization

AUTHENTICATION "คุณคือใคร?" พิสูจน์ตัวตน — login ด้วย email + password แล้วออก session
401 Unauthorized
ยังไม่ได้ login / token หมดอายุ → ระบบไม่รู้ว่าคุณเป็นใคร
AUTHORIZATION "คุณมีสิทธิ์ไหม?" ตรวจบทบาท — เป็น USER หรือ ADMIN ทำสิ่งนี้ได้หรือเปล่า
403 Forbidden
login แล้ว แต่ไม่มีสิทธิ์ → รู้ว่าคุณเป็นใคร แต่ห้ามทำ
⭐ AuthN ถาม "คุณคือใคร" · AuthZ ถาม "คุณทำสิ่งนี้ได้ไหม"
